# Portal 2 Menu Splashes Readme:

A Minecraft resource pack that replaces vanilla splash texts with portal 2 references.

<img width="349" height="91" alt="image" src="https://github.com/user-attachments/assets/3fa09be5-dc91-4081-9790-2d35f34f59ef" />

- Vanilla Splash Texts Replaced with Portal 2 references
- This resource pack is vanilla friendly
- Works on minecraft 1.21x

## Installation:
1. Download the zip from the releases page, modrinth or curseforge
2. Drop the pack into C:\Users\USER\AppData\Roaming\.minecraft\resourcepacks
3. Open Minecraft Java edition, click options, Resource Packs, and enable the pack by clicking the right facing arrow.

## Compatibility:

- Works on Minecraft Java Edition 1.21x
- Works with mods like Sodium and OptiFine
- On Minecraft versions 1.21.9+, the mod shows as incompatible even though it works fine
- The game must be restarted for changes to take effect.
